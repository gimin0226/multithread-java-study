## 목표

1. 자바의 스레드 모델과 JVM이 스레드를 관리하는 방식을 이해한다.
2. 교착상태(Deadlock), 경쟁 조건(Race Condition) 등 전형적인 동시성 문제를 실습과 함께 분석한다.
3. 운영체제의 동시성 이론과 자바 코드 구현을 연결하여 CS 이론과 코드 간의 겝을 메운다.
4. 멀티스레드 환경에서의 성능 최적화와 안전성을 동시에 고려하는 사고 방식을 기른다.
